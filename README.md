ProjectGen 
ok so basically this app generates cool project ideas for students!!
what it does 

u pick how much time u have, ur skill level and what kind of project u want
it uses AI to come up with a unique project idea just for u!!
dont like it? hit random idea for instant inspo 
saves ur fav ideas so u dont lose them
tracks stats like which category u pick the most n stuff
shows ur last 5 ideas on the side so u can go back to them
